# LifeGame - 人生管理游戏

一款将生活管理游戏化的 Android 应用，通过任务、属性、状态等机制帮助你追踪和提升自己的人生。

## 功能特性

### 属性系统
- 创建和管理自定义属性（IQ、EQ、体魄等）
- 属性拥有多个等级段位，达到条件自动触发庆祝
- 每种属性有独特的颜色标识

### 行动系统
- 定义每日行为/动作，带精力消耗或恢复
- 支持专注计时功能
- 行为可分组管理

### 任务系统
- 四种任务类型：每日、主线、支线、每周
- 支持属性目标和行为目标
- 任务完成/失败有奖励和惩罚机制

### 状态系统
- 临时 buff/debuff 效果
- 三种状态类型：
  - **周期生效**：按时间间隔自动增减属性
  - **加成**：百分比提升属性获取
  - **衰减**：百分比减少属性获取

### 日志系统
- 记录所有活动：行为执行、任务完成、等级提升等
- 支持锁定重要日志防止误删
- 日志导出功能

## 技术栈

| 技术 | 用途 |
|------|------|
| Kotlin | 开发语言 |
| MVVM + Clean Architecture | 应用架构 |
| Hilt | 依赖注入 |
| Room | 数据库 |
| Jetpack Navigation | 页面导航 |
| Kotlin Coroutines + Flow | 异步处理 |
| WorkManager | 后台任务 |

## 项目结构

```
app/src/main/java/com/example/lifegame/
├── data/
│   ├── dao/          # Room 数据访问对象
│   ├── entity/       # 数据库实体
│   └── local/        # 数据库配置
├── di/               # Hilt 依赖注入模块
├── repository/       # 仓库层
├── service/          # 后台服务
├── ui/
│   ├── attribute/    # 属性管理
│   ├── behavior/     # 行为管理
│   ├── focus/       # 专注计时
│   ├── log/         # 日志查看
│   └── quest/       # 任务管理
└── util/            # 工具类
```
